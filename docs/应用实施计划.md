# 宿舍管理系统 — C# WinForms 应用实施计划（第 2 份 / 共 2 份）

> **执行方式：** 应用代码由助手直接生成 `.cs` 文件到 `src\DormManagement\`，先用 `dotnet build` 编译验证，再由你 `dotnet run` / VS Code F5 运行测试。每个 Task 完成后构建必须通过、界面手测通过，再进入下一个。

**Goal:** 基于已建好的 `DormDB`，用 C# WinForms（.NET 10）实现三层架构桌面应用：登录、主界面、基础信息管理、入住/调宿/退宿、空床位/住宿/寝室名单查询、备份恢复。

**Architecture:** 三层 + 实体类。表示层（Forms）只管界面；数据访问层（`DBHelper`）统一用参数化命令访问 SQL Server；实体类（Models）传递数据。界面靠手写代码布局（VS Code 无拖拽设计器）。

**Tech Stack:** .NET 10（`net10.0-windows`）、WinForms、`Microsoft.Data.SqlClient`、`DataGridView` 绑定 `DataTable`。

**连接字符串：** `Server=localhost;Database=DormDB;Trusted_Connection=True;TrustServerCertificate=True;`

### Task A3：DBHelper（数据访问层核心）

**Files:** Create `DAL/DBHelper.cs`

- [ ] **完整代码：**
```csharp
using System.Data;
using Microsoft.Data.SqlClient;

namespace DormManagement.DAL
{
    public static class DBHelper
    {
        // 查询 → DataTable（绑定 DataGridView）
        public static DataTable QueryTable(string sql, params SqlParameter[] ps)
        {
            using var conn = new SqlConnection(AppConfig.ConnectionString);
            using var cmd = new SqlCommand(sql, conn);
            cmd.Parameters.AddRange(ps);
            using var da = new SqlDataAdapter(cmd);
            var dt = new DataTable();
            da.Fill(dt);
            return dt;
        }

        // 增删改 → 受影响行数
        public static int Execute(string sql, params SqlParameter[] ps)
        {
            using var conn = new SqlConnection(AppConfig.ConnectionString);
            using var cmd = new SqlCommand(sql, conn);
            cmd.Parameters.AddRange(ps);
            conn.Open();
            return cmd.ExecuteNonQuery();
        }

        // 标量
        public static object? Scalar(string sql, params SqlParameter[] ps)
        {
            using var conn = new SqlConnection(AppConfig.ConnectionString);
            using var cmd = new SqlCommand(sql, conn);
            cmd.Parameters.AddRange(ps);
            conn.Open();
            return cmd.ExecuteScalar();
        }

        // 调用存储过程 → DataTable
        public static DataTable ProcTable(string proc, params SqlParameter[] ps)
        {
            using var conn = new SqlConnection(AppConfig.ConnectionString);
            using var cmd = new SqlCommand(proc, conn) { CommandType = CommandType.StoredProcedure };
            cmd.Parameters.AddRange(ps);
            using var da = new SqlDataAdapter(cmd);
            var dt = new DataTable();
            da.Fill(dt);
            return dt;
        }
    }
}
```
> 全部走 `SqlParameter` 参数化 → 天然防 SQL 注入（报告安全性一节的论据）。

### Task A6：BuildingForm（宿舍楼，作为 CRUD 完整模板）

**Files:** Create `Forms/BuildingForm.cs`

- [ ] **完整模板代码**（其余 CRUD 窗体照此结构生成）：
```csharp
using System.Data;
using Microsoft.Data.SqlClient;
using DormManagement.DAL;

namespace DormManagement.Forms
{
    public class BuildingForm : Form
    {
        DataGridView grid = new() { Dock = DockStyle.Top, Height = 300, ReadOnly = true,
            SelectionMode = DataGridViewSelectionMode.FullRowSelect, AutoGenerateColumns = true };
        TextBox txtNo = new(), txtName = new(), txtGender = new();
        Button btnAdd = new() { Text = "新增" }, btnUpd = new() { Text = "修改" },
               btnDel = new() { Text = "删除" }, btnRefresh = new() { Text = "刷新" };

        public BuildingForm()
        {
            Text = "宿舍楼管理"; Width = 700; Height = 480;
            var panel = new FlowLayoutPanel { Dock = DockStyle.Bottom, Height = 120 };
            panel.Controls.AddRange(new Control[]{
                new Label{Text="楼号"}, txtNo, new Label{Text="楼名"}, txtName,
                new Label{Text="男/女"}, txtGender, btnAdd, btnUpd, btnDel, btnRefresh });
            Controls.Add(grid); Controls.Add(panel);

            Load += (_, _) => LoadData();
            btnRefresh.Click += (_, _) => LoadData();
            grid.SelectionChanged += (_, _) => FillInputs();
            btnAdd.Click += (_, _) => { DBHelper.Execute(
                "INSERT INTO Building(building_no,building_name,gender_type) VALUES(@n,@m,@g)",
                new SqlParameter("@n", txtNo.Text.Trim()),
                new SqlParameter("@m", (object?)txtName.Text ?? DBNull.Value),
                new SqlParameter("@g", (object?)txtGender.Text ?? DBNull.Value)); LoadData(); };
            btnUpd.Click += (_, _) => { if (CurrentId() is int id) { DBHelper.Execute(
                "UPDATE Building SET building_no=@n,building_name=@m,gender_type=@g WHERE building_id=@id",
                new SqlParameter("@n", txtNo.Text.Trim()),
                new SqlParameter("@m", (object?)txtName.Text ?? DBNull.Value),
                new SqlParameter("@g", (object?)txtGender.Text ?? DBNull.Value),
                new SqlParameter("@id", id)); LoadData(); } };
            btnDel.Click += (_, _) => { if (CurrentId() is int id &&
                MessageBox.Show("确认删除？","",MessageBoxButtons.YesNo)==DialogResult.Yes) {
                DBHelper.Execute("DELETE FROM Building WHERE building_id=@id", new SqlParameter("@id", id));
                LoadData(); } };
        }

        void LoadData() => grid.DataSource = DBHelper.QueryTable(
            "SELECT building_id 编号,building_no 楼号,building_name 楼名,gender_type 类别 FROM Building ORDER BY building_id");
        int? CurrentId() => grid.CurrentRow?.Cells["编号"].Value is int v ? v : null;
        void FillInputs()
        {
            if (grid.CurrentRow == null) return;
            txtNo.Text = grid.CurrentRow.Cells["楼号"].Value?.ToString();
            txtName.Text = grid.CurrentRow.Cells["楼名"].Value?.ToString();
            txtGender.Text = grid.CurrentRow.Cells["类别"].Value?.ToString();
        }
    }
}
```
- [ ] **验证**：菜单打开宿舍楼管理 → 列表显示 2 栋楼；新增"3号楼"→出现在列表；选中改名→更新；删除→消失。

```csharp

        new SqlParameter("@s", sid), new SqlParameter("@b", bid));
      DBHelper.Execute("INSERT INTO HousingLog(student_id,bed_id,op_type,operator) VALUES(@s,@b,'入住',@op)",
        new SqlParameter("@s", sid), new SqlParameter("@b", bid), new SqlParameter("@op", Operator));
      MessageBox.Show("入住成功"); }
catch (SqlException ex) when (ex.Number == 2627)   // 唯一约束冲突
```

## Phase 5：查询

### Task A13：EmptyBedQueryForm（空床位查询）
- [ ] **职责**：可选按楼/房筛选，`DataGridView` 列出空床：
```sql
SELECT b.building_no 楼号, r.room_no 房号, bd.bed_no 床号
FROM Bed bd JOIN Room r ON bd.room_id=r.room_id JOIN Building b ON r.building_id=b.building_id
WHERE bd.bed_id NOT IN (SELECT bed_id FROM Allocation)
```

## Phase 6：备份与恢复界面

### Task A16：BackupRestoreForm
- [ ] **职责**：三个按钮——
  - 「完整备份」→ `BACKUP DATABASE DormDB TO DISK=@path WITH INIT`（路径用备份目录 + 时间戳文件名）；
  - 「日志备份」→ `BACKUP LOG DormDB TO DISK=@path WITH INIT`；
  - 「时间点恢复」→ 一个 `DateTimePicker` 选时间，执行 `RESTORE DATABASE ... WITH NORECOVERY, REPLACE` + `RESTORE LOG ... WITH STOPAT=@t, RECOVERY`（先 `ALTER DATABASE DormDB SET SINGLE_USER WITH ROLLBACK IMMEDIATE`，恢复后 `SET MULTI_USER`）。
- [ ] **注意**：备份/恢复命令要连到 `master` 执行——本窗体单独用一个连到 master 的连接串（`Database=master`）。助手生成时单独处理。
- [ ] **验证**：点完整备份→backup 目录出现 .bak；选时间点恢复→数据回到该时刻（可复刻数据库 Task 9 的演示，但走界面）。
